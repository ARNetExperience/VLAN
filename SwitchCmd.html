Switch Command Guide


enable

This command switches the device into a mode where you can make configuration changes.

vlan 10 office_network

VLAN 10 added: Name: office_network

This command creates a new VLAN (Virtual Local Area Network) with an ID of 10 and gives it the name "office_network".

vlan 20 college_network

VLAN 20 added: Name: college_network

This command creates another VLAN with an ID of 20 and names it "college_network".

exit

APPLY completed.
 Exiting....

This command exits the configuration mode, meaning you've finished making changes and are ready to apply them.

show vlan

 VLAN   Name  Status 
 10   office_network   active 
 20   college_network   active 

This command shows the current VLANs on the switch and their status.
